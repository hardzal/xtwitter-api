# XTwitter-API

`npx tsc --init`

run
`npx tsx --env-file=.env --watch src/index.ts`
or
`npx tsc --noEmit && node --no-warnings=ExperimentalWarning --loader ts-node/esm src/index.ts`

check error
`npx tsc --noEmit`
